Color Changer Build Fractal Move Figure

This is information for users how to use my program

This is a special program for CG subject that will have different features and menus. So when you get to the main Fractal builder menu you can enter the amount of iteration you want want and build either Koch or a Dragon curve. The next menu is for changing colors of your fractal. So there are two colors model that you can use RGB and HSL, you have sliders for both of them and ca choose any color you want. More than that there is one more special slider for red color saturation that also can be used. There is also an opportunity of downloading image by pressing download button.
